## 개요

- 모던 자바스크립트 Deep Dive 책 완독 스터디
- 6주 완성

## 스터디 진행 방식

- 하루에 하루 분량 공부 ⇒ 자기 언어로 정리 ⇒ 핵심 질문 추출
- 주에 1회 미팅 (1시간 예상)
- 뽑기 (인당 2일분, 하나는 날려도 괜춘)
- 뽑힌 사람이 설명, 상대는 들으면서 정리 및 질문
